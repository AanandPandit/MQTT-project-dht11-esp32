The main.py code contains the code for esp32 and written in micropython 

This is for demo for communication between mqtt mosquitto and esp32 to control the device connected with esp32 WROOM

It allows to connect to wifi and it has features of controlling led1, led2, led3 from mqtt by subscribing the topic
It allows to publish temperaure and humidity data from dht11 to mqtt broker UI.
